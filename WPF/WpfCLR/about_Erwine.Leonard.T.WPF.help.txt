TOPIC
    about_Erwine.Leonard.T.WPF

SHORT DESCRIPTION
    Display WPF windows.

LONG DESCRIPTION
    Contains support for creating, dislaying and controlling WPF windows.

EXPORTED COMMANDS
    Following is a list of commands exported by this module:

    Assert-ValidXamlMarkup
        Asserts validity of XAML markup.

    Test-XamlMarkup
        Check validity of XAML markup.

    New-XamlWindowSource
        Create empty markup for new window.

    New-WpfWindow
        Create WPF Window proxy object.

    Show-WpfWindow
        Show a WPF Window.
